# Lims2 Autodeploy.

## How To Use

```
Usage:
  lims2 <command> [<args>...]
  lims2 -h | --help
  lims2 --version

Commands:
  up                   Create and start containers
  get-cron             Output crontab
  get-sphinx           Output sphinxsearch config
  update-cron          Output to /etc/cron.d/lims2 in container.

Options:
  -h --help     Show this screen.
  --version     Show version.
```
